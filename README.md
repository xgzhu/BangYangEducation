985&211大学生家教平台
===========================

基本信息
------------
本路径为谪仙科技下榜样教育微信小程序代码。该程序设计为允许中小学生及其家长联系大学生家教所用。家长可以发布请家教信息，大学生可以发布做家教信息。此外，相应合作的教育机构可以发布找实习信息给大学生，允许大学生到教育机构进行实习行授课。
